荣耀路由x1 增强版 修改 wi-fi 的 802.11n 频宽
荣耀路由x1 增强版 频宽是调制载波占据的频率范围，也是发送无线信号频率的标准。20mhz 穿透性较好，传输距离远，但传输速度最高只能达到 144mbps 带宽。40mhz 穿透性稍差，传输距离近，容易受到干扰，但传输速度却能达到 300mbps 带宽。因此，如果为了获得更大的传输速度，可以选用 40mhz。但是如果环境中，无线信号较多的话，则建议使用 20/40mhz 混合模式。步骤 1.电脑连接到路由器的 wi-fi（或者电脑连接到路由器的 lan 接口）。在浏览器地址栏中输入192.168.3.1登录路由器 web ui 管理页面。步骤 2.单击更多功能 > wi-fi 设置 > wi-fi 高级，在2.4g wi-fi 区域的 wi-fi 频宽设置中，选择想要设置的频宽。单击保存即可。
