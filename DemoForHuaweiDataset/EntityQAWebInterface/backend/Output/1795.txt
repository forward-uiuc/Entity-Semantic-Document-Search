荣耀路由x1 增强版 是否支持 5g wi-fi？如何判断当前连接的是 2.4g wi-fi 还是5g wi-fi
荣耀路由x1 增强版 支持 2.4g wi-fi 和 5g wi-fi，2.4g/5g wi-fi 名称默认相同，会自动为您连接更快的 wi-fi 信号。您可以通过以下方式查看手机当前连在2.4g wi-fi 还是 5g wi-fi：方式一：在手机上查看以华为手机为例：进入设置 > wlan，点击当前连接的 wi-fi 名称，可以查看。wi-fi 频率中显示 5ghz 表示当前连接的是 5g wi-fi，显示 2.4 ghz 表示当前连接的是 2.4g wi-fi。方式二：在 huawei hilink app 上查看步骤 1 手机连接到路由器的 wi-fi。步骤 2 打开 huawei hilink app 查看。方式三：在电脑 web ui 管理页面查看步骤 1 电脑连接到路由器的 lan 接口，在浏览器地址栏中输入192.168.3.1登录路由器 web ui 管理页面。步骤 2 单击终端管理可以查看。
