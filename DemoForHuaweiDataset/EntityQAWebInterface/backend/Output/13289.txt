隐藏荣耀路由x1 增强版 的 wi-fi 名称
荣耀路由x1 增强版 隐藏 wi-fi 名称后，手机或电脑等无线设备将搜索不到路由器的 wi-fi。手机或电脑连接这个 wi-fi 的时候必须手动输入正确的 wi-fi 名称。这样可以在一定程度上增强 wi-fi 的安全性。1.电脑连接到路由器的 wi-fi（或者电脑连接到路由器的 lan 接口）。在浏览器地址栏中输入192.168.3.1登录路由器 web ui 管理页面。2.单击更多功能 > wi-fi 设置 > wi-fi 高级。在2.4g wi-fi 和 5g wi-fi 区域中，选择 wi-fi 隐身开启，即可分别隐藏路由器的 2.4g或 5g wi-fi 名称。
